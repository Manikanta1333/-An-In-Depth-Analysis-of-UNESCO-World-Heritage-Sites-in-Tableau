An In-Depth Analysis of UNESCO World Heritage Sites in Tableau

"Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites (2019)" is a comprehensive project aimed at exploring the rich dataset of UNESCO World Heritage Sites using Tableau.

Scenarios:

  • Heritage Sites by Country: Treemap based on the number of sites per country.
  • Sites at Risk: Pie chart dividing ‘In Danger’ vs ‘Not in Danger’ sites.
  • Regional Inscription Trends: Line chart showing inscription trends over time by region.

About the Project

This project explores UNESCO World Heritage Sites using Tableau to visualize patterns, risks, and historical trends. It includes treemaps, pie charts, and line charts based on region, status, and country.

The visualizations help improve awareness and support global preservation efforts.
